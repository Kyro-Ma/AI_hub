## 每日 AI 资讯更新说明

本项目**每日更新 AI 相关新闻**，并保持：
1) **按年月归档**（便于历史检索）
2) **根目录今日汇总**：`今日AI资讯.md`（便于读者快速查看最新内容）

## 目录结构

- `./今日AI资讯.md`  
  - 永远保存“今天”的最新资讯
    
- `./YYYY/MM/`（按年/月归档目录）  
  - 存放每天的历史记录文件，例如：
    - `./2026/02/2026-02-04.md`
    - `./2026/02/2026-02-05.md`
